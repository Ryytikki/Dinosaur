Dino_game.exe created by MyTriforce (Morgan) and Ryytikki (Tom)

Controls
========

Movement: WASD
Swap characters: 1/2
Interact: Space
Exit: Esc

Description
===========

Use the dinosaurs to (once the level is fully built) escape

Triceratops: 		Can push boulders
			Can activate buttons
			Can carry other dinosaurs
			 
Velociraptor: 
			Jumps considerably higher (the ability to "fly" is just to help me move around the map better when testing)
			Can ride on Triceratops' back (sorta)
			Can carry bombs
			Can activate wall buttons
